# Thinkpair

A simple Flutter application for generating creative word pairs and managing them through favorites and history.

---

## Features

- Generate random English word pairs
- Like and unlike word pairs
- View all liked words in the Favorites section
- View previously generated words in the History section
- Navigate between pages using NavigationRail
- Responsive layout for different screen sizes

---
## How to Run

1. Make sure Flutter is installed  
2. Clone this repository  
3. Run the following commands:

```
bash
flutter pub get
flutter run
